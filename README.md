# Financial Data Warehouse Migration to AWS – AUTOX

This project demonstrates migrating a legacy on-premise financial data warehouse to AWS using S3, Redshift, Snowflake, Spark, Glue, and Airflow. It enables scalable data warehousing on distributed systems and reduces infrastructure management effort.

## Features
- Core ETL pipelines using Spark, Glue, and Airflow
- Migration to S3, Redshift, and Snowflake
- Modular architecture for scalable financial analytics
- Automated orchestration and monitoring

## Technologies
- Python, PySpark
- Apache Airflow
- AWS S3, Redshift, Glue
- Snowflake
